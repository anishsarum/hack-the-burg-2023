### An Amazing Project

This is going to be an amazing project for HTB IX.

## 1. Setup

1. Run server-side FastAPI app in one terminal...

```
cd backend
python3 -m venv env
source env/bin/activate
pip install -r requirements.txt
python main.py
```

2. Run client-side React app in different terminal...

```
cd frontend
npm install
npm run start
```
